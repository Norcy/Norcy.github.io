新型冠状病毒城市趋势图（202002）

地狱空荡荡，魔鬼在人间

由于一直很关注深圳市的疫情趋势，想知道它的确诊人数的变化趋势是如何的，有助于做决定或提前做好各种准

苦于找不到相关的工具/网站，所以 2 月 6 号下午心血来潮，自己动手造了个小轮子，可以方便快速查询城市疫情的发展趋势

不过现在疫情缓解了，很多 API 接口也废除了，所以不能用了（202005 注）

Github 地址：nCov-City
